# 프로젝트 개요
스프링부트의 기본적인 사용방법을 익히고자 진행한 쇼핑몰 프로젝트

**개발 인원**: 개인 프로젝트  
**개발 기간**: 11월 3일 ~ 12월 23일

# 사용 기술
* Spring boot
* JAVA
* JPA
* Spring Security
* QueryDSL
* MariaDB

# 주요 기능
* 상품, 댓글, 사용자 CRUD
* 상품 검색 및 결제
* 장바구니 기능
* 로그인
